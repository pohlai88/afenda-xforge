# React Wrap Balancer

`react-wrap-balancer` improves text wrapping so titles stay easier to read across
different viewport sizes.

## Installation

Install `react-wrap-balancer` with your preferred package manager:

```bash
npm install react-wrap-balancer
```

```bash
pnpm add react-wrap-balancer
```

```bash
yarn add react-wrap-balancer
```

```bash
bun add react-wrap-balancer
```

## Usage

Wrap long titles with `Balancer` to make the line breaks more balanced:

```tsx title="my-component.tsx"
import { Balancer } from "react-wrap-balancer";

function MyComponent() {
  return (
    <div>
      <Balancer>This is a title that is too long to fit in one line.</Balancer>
    </div>
  );
}
```

## Benefits

- Improved readability for long headings and titles.
- Easy to integrate into an existing React application.

For more information and detailed documentation, visit the
[react-wrap-balancer website](https://react-wrap-balancer.vercel.app/).
