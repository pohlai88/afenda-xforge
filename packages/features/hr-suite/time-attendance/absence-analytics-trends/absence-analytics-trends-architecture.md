# Absence Analytics & Trends

## Definition

**Absence Analytics & Trends is the HRM function that analyzes employee absence patterns, leave behavior, attendance exceptions, unplanned absence, recurring absence, departmental absence rates, and workforce availability trends to identify risk, reduce disruption, and support better workforce planning.**

---

# Absence Analytics & Trends Includes

| Area                            | What It Covers                                                                                |
| ------------------------------- | --------------------------------------------------------------------------------------------- |
| **Absence Pattern Analysis**    | Repeated absence, frequent short leave, Monday/Friday absence, post-holiday absence           |
| **Unplanned Leave Analysis**    | Emergency leave, unpaid leave, unapproved absence, no-show, last-minute absence               |
| **Leave Trend Analysis**        | Leave usage by employee, team, department, location, manager, period                          |
| **Attendance Exception Trends** | Late arrival, early departure, missing punch, absence without approval                        |
| **Team Absence View**           | Absence rate by team, department, branch, legal entity, manager                               |
| **Period-Based Analysis**       | Daily, weekly, monthly, quarterly, yearly absence trends                                      |
| **Leave Type Analysis**         | Annual leave, medical leave, unpaid leave, emergency leave, sick leave, hospitalization leave |
| **High-Risk Absence Detection** | Excessive absence, repeated absence, abnormal absence pattern, policy breach indicator        |
| **Absence Rate Calculation**    | Absence percentage, lost workdays, absence frequency, absence duration                        |
| **Workforce Availability View** | Available employees, unavailable employees, capacity risk, coverage risk                      |
| **Manager Dashboard**           | Team absence summary, exception list, risk employees, trend indicators                        |
| **HR Analytics Dashboard**      | Absence heatmap, department ranking, location comparison, trend movement                      |
| **Absence Forecast Reference**  | Expected absence pressure, seasonal absence trend, workforce coverage warning                 |
| **Policy Compliance Indicator** | Breach of absence policy, excessive unpaid leave, repeated late attendance                    |
| **Corrective Action Reference** | Coaching reference, HR review reference, attendance improvement plan reference                |
| **Payroll Impact Reference**    | Unpaid leave deduction reference, absence deduction reference, lateness deduction reference   |
| **Reporting**                   | Absence trend report, absenteeism report, exception report, leave utilization report          |
| **Audit Trail**                 | Analytics generated by, reviewed by, exported by, action taken by, timestamp                  |

# Absence Analytics & Trends Requirement Statement

| Requirement                    | Description                                                                                                                                                                                                                                                                                                                        |
| ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Absence Analytics & Trends** | Analyzes employee absence patterns across teams, departments, locations, managers, and time periods to identify recurring absence, unplanned leave, attendance exceptions, workforce availability risk, policy breach indicators, payroll impact references, corrective action references, reports, dashboards, and audit history. |

---

# Enterprise Functional Requirements

| Code            | Requirement                                                                                                                                |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **HRM-AAT-001** | System shall analyze employee absence data by employee, team, department, manager, location, legal entity, and period.                     |
| **HRM-AAT-002** | System shall calculate absence rate by employee, team, department, location, and period.                                                   |
| **HRM-AAT-003** | System shall calculate absence frequency.                                                                                                  |
| **HRM-AAT-004** | System shall calculate total lost workdays due to absence.                                                                                 |
| **HRM-AAT-005** | System shall analyze absence duration by leave type and period.                                                                            |
| **HRM-AAT-006** | System shall identify unplanned leave trends.                                                                                              |
| **HRM-AAT-007** | System shall identify repeated short absence patterns.                                                                                     |
| **HRM-AAT-008** | System shall identify frequent Monday, Friday, pre-holiday, or post-holiday absence patterns.                                              |
| **HRM-AAT-009** | System shall identify employees with excessive absence based on configured thresholds.                                                     |
| **HRM-AAT-010** | System shall identify departments or teams with high absence rates.                                                                        |
| **HRM-AAT-011** | System shall identify attendance exception trends including late arrival, early departure, absence, and missing punch.                     |
| **HRM-AAT-012** | System shall compare planned leave against unplanned absence.                                                                              |
| **HRM-AAT-013** | System shall compare absence trends across departments, locations, managers, and employee groups.                                          |
| **HRM-AAT-014** | System shall provide workforce availability indicators.                                                                                    |
| **HRM-AAT-015** | System shall flag workforce coverage risk caused by high absence levels.                                                                   |
| **HRM-AAT-016** | System shall provide absence heatmap by date, team, department, location, or leave type.                                                   |
| **HRM-AAT-017** | System shall provide trend movement indicators for improving, worsening, or stable absence patterns.                                       |
| **HRM-AAT-018** | System shall support configurable absence risk thresholds.                                                                                 |
| **HRM-AAT-019** | System shall classify absence risk as normal, watch, at risk, high risk, or critical.                                                      |
| **HRM-AAT-020** | System shall expose absence risk indicators to authorized HR users and managers.                                                           |
| **HRM-AAT-021** | System shall link absence insights to corrective action references where required.                                                         |
| **HRM-AAT-022** | System shall expose unpaid absence and attendance deduction references to Payroll Processing through Leave & Attendance Management.        |
| **HRM-AAT-023** | System shall support absence trend reports by employee, department, manager, location, legal entity, leave type, and period.               |
| **HRM-AAT-024** | System shall support export of absence analytics reports by authorized users.                                                              |
| **HRM-AAT-025** | System shall restrict access to absence analytics based on employee, manager, HR, payroll, compliance, and auditor permissions.            |
| **HRM-AAT-026** | System shall mask sensitive absence reasons where the user is not authorized to view them.                                                 |
| **HRM-AAT-027** | System shall notify HR or managers when absence risk exceeds configured thresholds.                                                        |
| **HRM-AAT-028** | System shall preserve historical absence analytics snapshots by period.                                                                    |
| **HRM-AAT-029** | System shall maintain audit trail for analytics generation, threshold change, report export, risk review, and corrective action reference. |
