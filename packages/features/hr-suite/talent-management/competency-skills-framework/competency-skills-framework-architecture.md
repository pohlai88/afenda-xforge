# Competency & Skills Framework

## Definition

**Competency & Skills Framework is the HRM function that defines, organizes, maps, assesses, and governs organizational competencies, job-required skills, employee skill profiles, proficiency levels, skill gaps, competency requirements, and development planning references.**

---

# Competency & Skills Framework Includes

| Area                               | What It Covers                                                                                |
| ---------------------------------- | --------------------------------------------------------------------------------------------- |
| **Competency Library**             | Core competencies, leadership competencies, technical competencies, behavioral competencies   |
| **Skill Library**                  | Technical skills, soft skills, functional skills, digital skills, operational skills          |
| **Competency Categories**          | Leadership, communication, problem-solving, customer service, compliance, technical expertise |
| **Skill Categories**               | Finance, HR, sales, operations, engineering, IT, manufacturing, safety, quality               |
| **Proficiency Levels**             | Beginner, working knowledge, competent, advanced, expert                                      |
| **Role-Based Competency Mapping**  | Required competencies by role, job family, grade, position, department                        |
| **Job Skill Requirements**         | Mandatory skills, preferred skills, critical skills, certification-linked skills              |
| **Employee Skill Profile**         | Employee skills, current proficiency, evidence, last assessed date                            |
| **Skill Gap Analysis**             | Required level vs actual level, missing skills, development priority                          |
| **Competency Assessment**          | Self-assessment, manager assessment, HR assessment, evidence-based assessment                 |
| **Development Planning Reference** | Recommended training, coaching, mentoring, certification, stretch assignment                  |
| **Career Path Reference**          | Skills required for next role, promotion readiness, lateral movement readiness                |
| **Succession Reference**           | Successor readiness, critical role skill gaps, leadership pipeline capability                 |
| **Training Linkage**               | Course mapping to competency or skill gap                                                     |
| **Certification Linkage**          | Required license, qualification, professional certificate                                     |
| **Reporting**                      | Skill inventory report, competency gap report, readiness report, workforce capability report  |
| **Audit Trail**                    | Created by, assessed by, reviewed by, approved by, changed by, timestamp                      |

---

# Competency & Skills Framework Does Not Include

| Excluded Area                   | Owned By                                          |
| ------------------------------- | ------------------------------------------------- |
| Employee master profile         | Employee Records Management                       |
| Job position ownership          | Organizational Chart & Hierarchy / Job Management |
| Performance review scoring      | Performance Appraisals                            |
| Training course delivery        | Learning Management System                        |
| Training program administration | Training & Development                            |
| Certification document storage  | Document Management                               |
| Compensation adjustment         | Compensation Planning & Modeling                  |
| Bonus calculation               | Bonus & Incentive Management                      |
| Payroll calculation             | Payroll Processing                                |
| Recruitment pipeline            | Recruitment & Applicant Tracking                  |
| Succession nomination decision  | Succession Planning                               |
| Employee promotion workflow     | Employee Lifecycle Management                     |
| Workforce planning budget       | Workforce Planning / Finance                      |
| Compliance rule monitoring      | Compliance & Regulatory Tracking                  |

# Competency & Skills Framework Requirement Statement

| Requirement                       | Description                                                                                                                                                                                                                                                                                                                                                                        |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Competency & Skills Framework** | Defines organizational competencies and skill libraries, maps required competencies and skills to roles, jobs, grades, positions, and departments, maintains employee skill profiles, assesses proficiency levels, identifies skill gaps, links gaps to development actions, supports career and succession readiness, provides capability reporting, and maintains audit history. |

---

# Enterprise Functional Requirements

| Code            | Requirement                                                                                                                                                          |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **HRM-CSF-001** | System shall create and maintain a competency library.                                                                                                               |
| **HRM-CSF-002** | System shall create and maintain a skill library.                                                                                                                    |
| **HRM-CSF-003** | System shall classify competencies by category such as core, leadership, technical, behavioral, functional, safety, and compliance.                                  |
| **HRM-CSF-004** | System shall classify skills by job family, department, function, role, or capability domain.                                                                        |
| **HRM-CSF-005** | System shall define proficiency levels for competencies and skills.                                                                                                  |
| **HRM-CSF-006** | System shall support proficiency scale descriptions and assessment criteria.                                                                                         |
| **HRM-CSF-007** | System shall map required competencies to job roles, job families, grades, positions, departments, or legal entities.                                                |
| **HRM-CSF-008** | System shall map required skills to job roles, job families, grades, positions, departments, or legal entities.                                                      |
| **HRM-CSF-009** | System shall classify required skills as mandatory, preferred, critical, or optional.                                                                                |
| **HRM-CSF-010** | System shall define required proficiency level for each mapped competency or skill.                                                                                  |
| **HRM-CSF-011** | System shall maintain employee competency profiles.                                                                                                                  |
| **HRM-CSF-012** | System shall maintain employee skill profiles.                                                                                                                       |
| **HRM-CSF-013** | System shall record employee current proficiency level for each competency or skill.                                                                                 |
| **HRM-CSF-014** | System shall support self-assessment of skills and competencies where enabled.                                                                                       |
| **HRM-CSF-015** | System shall support manager assessment of employee skills and competencies.                                                                                         |
| **HRM-CSF-016** | System shall support HR or expert validation of skill and competency assessments where required.                                                                     |
| **HRM-CSF-017** | System shall record assessment evidence, source, assessment date, assessor, and confidence level.                                                                    |
| **HRM-CSF-018** | System shall calculate skill gaps by comparing required proficiency against current proficiency.                                                                     |
| **HRM-CSF-019** | System shall calculate competency gaps by comparing required competency level against current competency level.                                                      |
| **HRM-CSF-020** | System shall classify gaps by severity, priority, role impact, and development urgency.                                                                              |
| **HRM-CSF-021** | System shall recommend development actions based on identified skill or competency gaps.                                                                             |
| **HRM-CSF-022** | System shall link skill gaps to training courses, learning paths, certifications, coaching, or development plans.                                                    |
| **HRM-CSF-023** | System shall expose competency and skill gaps to Training & Development.                                                                                             |
| **HRM-CSF-024** | System shall expose learning recommendations to Learning Management System where enabled.                                                                            |
| **HRM-CSF-025** | System shall expose competency references to Performance Appraisals where authorized.                                                                                |
| **HRM-CSF-026** | System shall expose readiness indicators to Succession Planning where authorized.                                                                                    |
| **HRM-CSF-027** | System shall support career path skill requirement comparison.                                                                                                       |
| **HRM-CSF-028** | System shall identify employees who match required skills for a role, project, or critical position.                                                                 |
| **HRM-CSF-029** | System shall provide competency and skill reports by employee, role, department, job family, grade, location, and proficiency level.                                 |
| **HRM-CSF-030** | System shall restrict skill, competency, assessment, and readiness data based on employee, manager, HR, leadership, and auditor permissions.                         |
| **HRM-CSF-031** | System shall maintain audit trail for competency setup, skill setup, role mapping, assessment, validation, gap analysis, recommendation, and profile update actions. |

|  15 | Competency gaps are calculated by comparing required level against current level.                                                                   |
|  16 | Gaps can be classified by severity, priority, role impact, and development urgency.                                                                 |
|  17 | Development actions can be recommended based on identified gaps.                                                                                    |

|  21 | Competency references are available to Performance Appraisals where authorized.                                                                     |
|  22 | Readiness indicators are available to Succession Planning where authorized.                                                                         |
|  23 | Career path comparison can show skills required for next role.                                                                                      |
