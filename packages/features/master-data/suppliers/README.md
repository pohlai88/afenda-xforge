# `@repo/features-master-data-suppliers`

Master-data feature package for supplier governance.

The package follows the canonical XForge master-data shape:

- `contract.ts` for schemas and route contracts
- `manifest.ts` for shell registration metadata
- `metadata.ts` for declarative table hints
- `server.ts` for the approved public server entrypoint
- `shared/` and `execution/` for feature-local primitives
